Overview of Projects

Project GoldStars X

Project GoldStars X is a full-screen Java program designed to appear as a virtual operating system. It includes many features, such as an agenda, a calculator, a notes program, a photo viewer, and more! It can serve as a useful productivity tool, as well as a a project to help people trying to learn and/or experiment with Java.

Project GoldStars Calculator S

Project GoldStars Calculator S is a convenient and robust Java calculator. It features a wide range of calculation options and can be customized with different themes.

Project GoldStars Calculator C

Project GoldStars Calculator C is a simple yet useful Java command-line calculator.

Project Hello7000

Project Hello7000 is a chatbot developed in Java.

Project CyanSky X

Project CyanSky X is a simple C program with a few features.
